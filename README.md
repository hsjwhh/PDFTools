### PDF 工具练习

- 使用 itext7 进行 pdf 文件操作
- 按照说明，需要安装两个包，切记。
```shell
dotnet add package itext --version 8.0.2
dotnet add package itext.bouncy-castle-adapter --version 8.0.2
```
